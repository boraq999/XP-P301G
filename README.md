# مشروع الطباعة الصامتة (XP-P301G)

هذا المشروع يسهل عملية الطباعة الحرارية من المتصفح مباشرة لطابعة XP-P301G عبر طريقتين: السيرفر المحلي (Windows) أو البلوتوث (Web Bluetooth).

## الاقتراحات الحالية للتحسين:

1.  **دعم اللغة العربية في الطباعة النصية:** تحسين جودة النصوص بدعم مكتبات تحويل النص العربي إلى ESC/POS مباشرة بدلاً من الاعتماد الكلي على الصور، مما يزيد من سرعة الطباعة.
2.  **إضافة واجهة إعدادات للطابعة:** توفير واجهة مستخدم (UI) تتيح تغيير اسم الطابعة المشتركة وماكينات الطباعة دون الحاجة لتعديل الكود المصدري.
3.  **تطوير واجهة البلوتوث (قيد التنفيذ):** جعل النظام يعمل بالكامل من المتصفح (خاصة للهواتف) عبر بروتوكول Web Bluetooth، مع دعم طباعة الصور والباركود دون الحاجة لسيرفر Node.js.
